How to Set Up Hover Redirects
  1. Go to hover.com and sign in
  2. Click the domain to configure
  3. Click the Forward tab (or URL Forwarding)
  4. Click Add a Forward
  5. Select Permanent Redirect (301)
  6. Paste the Amazon URL from the Domain Redirects tab
  7. Save and repeat for all 10 domains
  8. Test in browser after setup (DNS takes 1-2 hours)
How to Add HTML to Magento Product Pages
  1. Go to the Product Pages tab
  2. Click a product to expand it
  3. Click Copy HTML
  4. In Magento admin, open the product
  5. Switch description to HTML/Source mode
  6. Paste above or below existing content - do NOT replace it
  7. Save
